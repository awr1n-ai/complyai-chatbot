# 🚀 ComplyAI – AI Governance Sandbox for Banks

## 🧠 Problem

Banks fail to deploy AI because models are not explainable and fail regulatory checks.

## 💡 Solution

ComplyAI allows banks to:

* Upload AI models
* Test against regulations
* Get explainable results using SHAP
* Generate audit-ready reports

## ⚙️ Tech Stack

* Frontend: React
* Backend: FastAPI
* Database: PostgreSQL
* AI Explainability: SHAP

## 🚀 Features

* Model upload
* Compliance testing
* Explainable AI results
* PDF audit reports

## 🎯 Impact

* Faster AI approval
* Transparent decisions
* Reduced bias
