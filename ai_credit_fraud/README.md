# AI Credit Scoring & Fraud Detection Model

This folder contains a built-out pipeline for dual AI models.

## Features
1. Synthetic data generation simulating user profiles and transaction events.
2. Random Forest Regressor to predict the user's Credit Score.
3. Random Forest Classifier to detect whether a transaction is Fraudulent.

### How to Run:
```bash
pip install -r requirements.txt
python main.py
```
